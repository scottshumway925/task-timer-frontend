# Canvas Assignment Timer

Time users as they complete assignments, and display data about the times that have been submitted, such as mean, median, and mode.

## Instructions for Build and Use

Steps to build and/or run the software:

1. run ```npm run test``` to build the bundle if any changes have been made
2. Go into your extension page and click the load unpacked button
3. Select this folder

Instructions for using the software:

1. Go to any assignment/ quiz on canvas
2. You can see any data that has always been inputted
3. Use the timer to time yourself
4. Submit your time when you've finished the assignment

## Development Environment 

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* run ```npm install``` to pull all dependencies
* ```npm run test``` to build the bundle
